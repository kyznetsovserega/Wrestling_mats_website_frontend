<h1 align="center">Prospect Boxing — Landing Page</h1>

<p align="center">
Лендинг для продажи борцовских и боксёрских покрытий.  
Верстка выполнена на **HTML + CSS (BEM)** и **JavaScript** .
</p>

---

## Возможности интерфейса

### Hero-блок
- Parallax-эффект по движению мыши  
- Динамическая виньетка + шумовой слой  
- Плавная анимация появления элементов  
- Автоматический mobile-layout  
- Корректный поворот фона в портретном режиме

---

### Навигация
- Sticky-header с изменением внешнего вида при скролле  
- Desktop-меню + отдельная Mobile-NAV  
- Анимированный бургер (content: "☰" → "✖") 
- Полноценный отдельный mobile-layer
- Автоматическое скрытие Social Bar

---

### Каталог покрытий (MATS)
- Сетка `auto-fit` + `minmax()` — флюидная адаптивная сетка  
- Hover-эффект увеличения  
- Мобильный режим: картинка слева, текст справа  
- Интерактивная кнопка расширения карточки

---
### Fasteners (типы крепления)
- Интерактивная схема с hotspot-кнопками  
- Активация кругового просмотра (circle-image)
- Детализированные панели:
  - Карманы (конверт)
  - Липучки (Velcro)
  - Люверсы
- Полностью адаптивная работа табов

---

### Product Sheet (Bottom-Sheet Modal)
- Модальное окно снизу (iOS-style bottom-sheet) 
- Slice preview (просмотр «среза» покрытия) 
- Выбор фиксированных размеров
- Готовые размеры и кастомный ввод Ш×Д  
- Автоматический расчёт стоимости  
- Полноэкранный mobile-режим

---

### Галерея
- 3 карточки на десктопе  
- Мобильный режим: горизонтальный scroll-slider  
- Custom scrollbar  
- Без внешних библиотек  

---

## Технологии

- **HTML5 + CSS3 (BEM)**  
- **JavaScript ES6+**  
- Анимации без библиотек  
- Производительные слои (transform / opacity)  
- Адаптивные сетки и fluid-layout

---

## JavaScript
- Модульный подход
- Минимум DOM-операций
- Точные event-listeners
- Никаких зависимостей / библиотек

---

## Производительность
- Все тяжёлые эффекты через GPU-слои
- Ленивая инициализация анимаций
- Минимизация redraw / reflow
- Оптимизация изображений
